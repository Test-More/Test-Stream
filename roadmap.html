Experimental

Experimental Phase (current)

  • Test-Stream released to cpan independant of Test::Builder
  • Test-Stream reserves the right to break anything in its own dist in this phase
  • Test-Simple dev releases that use Test::Stream
  • Nothing production should depend on Test::Stream during this phase
The purpose of this phase is to allow people to use, and provide feedback on Test::Stream. It is a time of heavy churn, we want to be able to change things as needed.
Punchlist

Punchlist Phase (future)

This phase will start (restart) when Test::Stream seems to be ready. In this phase focus will go into completing the punchlist set at the 2015 perl-qa hackathon.
Stable

Stable Phase (future)

A stable Test-Simple will be released either with Test::Stream bundled, or with it listed as a dep.
